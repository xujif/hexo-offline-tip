## show hexo offline tips (in pwa mode)  

### install
```
npm install hexo-offline-tip
```
or with yarn
```
yarn add hexo-offline-tip
```

### config
```
offline_tips:
  enable: true 
  options:                                        # optional
    toastText: 'your network is offline',         # optional tip text
    toastStyle:                                   # optional style eg. 'color: red'
```
